MLB Data Analysis Project

As a part of my data visualization class at Syracuse University, we were tasked with finding a data set that interested us and using it to tell a story to an audience. I decided to look into the MLB salary cap and the problems that stem from the league that is becoming ever more "Pay to Play". This project was created with R (for the data cleaning and analysis) and Adobe Illustrator for the poster portion of the project.

Storm Data Analysis Project

In the same Data Visualization class, we had a week that was dedicated to do a similar project as the one above, yet it was with a group. Some of the members in our group had experienced extreme storms in their life and the effects of that, so we decided to dive deeper into that subject. We used packages in R like ggmap to inspect the true costs of these storms in America and more specifically Texas. This was an extremely fun project to collaborate and learn from my colleagues.

Python Game Creation

In one of my Python Coding classes during Junior year, we got the oppurtunity to have some creative freedom with a program that we created. With my background in athletics, I decided to create an interactive quiz game that can be played by anyone. It leveraged the use of Web APIs to reference databases to create an interative quiz with real time data. How the game works is you select a team, and what year you want and you try to guess the leaders in different stats listed. Then you get results displayed, as well as sent to an email if you so chose. Clearly this is a beta version of an application, but a good way to see a basic python program.
